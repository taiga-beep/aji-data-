# OpenRouter Price Data

LLM API prices fetched daily from [OpenRouter API](https://openrouter.ai/api/v1/models) via GitHub Actions.

Data is served via GitHub Pages at:
```
https://{username}.github.io/openrouter-price-data/data/summary.json
```

Used by [AI Japan Index — LLM API Price Tracker](https://ai-japan-index.com/openrouter-price-tracker/).

## Schedule
- Daily at 16:00 JST (07:00 UTC)
- Manual trigger available via Actions tab
